Rotation curves files from

de Blok et al. Astronomical Journal 2008

For plotting and analysis purposes, use Columns (1), (2), (7)

Column 1: Radius in arcsec
Column 2: Rotation Velocity (km/s)
Column 3: Scatter in observed rotation velocity along ring
Column 4: Formal error in tilted ring rotation velocity fit
Column 5: Rotation velocity approaching side velocity field
Column 6: Rotation velocity receding side velocity field
Column 7: Combined error <--- preferred error measure
Column 8: Position Angle
Column 9: Inclination
Column 10: Systemic velocity
